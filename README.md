# Sistema de Inventario de Biblioteca con RMI

Bienvenido al repositorio del proyecto para el sistema de inventario de biblioteca con las llamadas 
a procedimientos remotos.

Proyecto para la Experiencia educativa de desarrollo de sistemas en red

## Estructura del proyecto 

El proyecto contiene 2 carpetas , las cuales son:

- `src`: La carpeta con el codigo 
- `lib`: La carpeta con las dependencias

Tambien, las salidas de compilacion seran generadas en la carpeta `bin` de manera predeterminada

> If you want to customize the folder structure, open `.vscode/settings.json` and update the related settings there.
