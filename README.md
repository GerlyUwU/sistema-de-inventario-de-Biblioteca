# Sistema de Inventario de Biblioteca con RMI

Bienvenido al repositorio del proyecto para el sistema de inventario de biblioteca con las llamadas 
a procedimientos remotos.

Proyecto para la Experiencia educativa de desarrollo de sistemas en red

## Estructura del proyecto 

El proyecto contiene 2 carpetas , las cuales son:

- `cliente_practica`: La carpeta con el codigo de la vista del cliente segun la arquitectura RPC 
- `Servidor_practica`: La carpeta con el codigo del servidor segun la arquitectura RPC
